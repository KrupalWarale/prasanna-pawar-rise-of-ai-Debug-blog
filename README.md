# Rise of AI in Debugging

A modern web project exploring the evolution and future of AI-powered debugging tools.

## Features

- Interactive timeline of AI debugging evolution
- 25 comprehensive chapters on AI debugging topics
- Responsive design with smooth animations
- Modern glassmorphism UI effects

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Navigate through chapters and explore the content

## Structure

- `index.html` - Main landing page
- `chapters/` - Individual chapter pages
- `style.css` - Main stylesheet
- `app.js` - JavaScript functionality
- `images/` - Project images and assets
- `videos/` - Background videos

## Technologies

- HTML5
- CSS3 (with glassmorphism effects)
- Vanilla JavaScript
- AOS (Animate On Scroll) library
